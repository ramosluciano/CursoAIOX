## 💡 Principais Insights

### 1. Claude Code é Camadas
```
L1: Global settings (~/.claude/)
L2: Project config (.claude/)
L3: Session memory
L4: Runtime execution
```

### 2. Context é Limitado
- Sempre há trade-off entre knowledge e speed
- Use rules, memory, e commands para gerenciar
- Limpe contexto regularmente com /clear

### 3. Permission Modes são seu Superpower
- Ask: Seguro, interativo
- Plan: Explore sem risco
- Auto: Rápido mas controlado
- Shift+Tab para ciclar

### 4. MCP Conecta Tudo
- GitHub, Slack, Docker, databases
- Estenda funcionalidade
- Integre com tools existentes

### 5. Automação é Chave
- Rules para regras específicas
- Commands para workflows
- Hooks para garantias determinísticas
- Tasks para coordenação

## 🛠️ Setup Mínimo

Para começar agora mesmo:

```bash
# 1. Instalar
npm install -g @anthropic-ai/claude-code

# 2. Autenticar
claude auth

# 3. Criar CLAUDE.md
/init

# 4. Customizar
# Editar .claude/CLAUDE.md com seu contexto

# 5. Começar!
claude "your first prompt"
```

**Tempo total: 10 minutos**
